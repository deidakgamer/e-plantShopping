# e-plantShopping

## 🌱 Proyecto: e-plantShopping

Esta es una aplicación web desarrollada con React y Vite que permite a los usuarios explorar y comprar plantas en línea.

## 🚀 Características
- Visualización de plantas
- Carrito de compras
- Interfaz moderna con React

## 🛠️ Tecnologías usadas
- React
- Vite
- JavaScript
- CSS
